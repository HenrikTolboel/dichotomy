Download
--------
depend via Maven:

```xml
<dependency>
  <groupId>net.xyzsd</groupId>
  <artifactId>dichotomy</artifactId>
  <version>0.9</version>
  <type>module</type>
</dependency>
```

or Gradle:
```kotlin
implementation("net.xyzsd:dichotomy:0.9")
```
